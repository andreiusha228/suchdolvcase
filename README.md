# Suchdol v čase

Modern, minimalist single-page website for the finished documentary project "Suchdol v čase".

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js App Router pages and layout
- `components/` - React components (Navbar, Hero, OProjektu, Film, Kontakt, Footer)
- `app/globals.css` - Global styles and Tailwind imports

## Features

- Minimalist, cinematic design with black background
- Smooth scroll navigation
- Responsive layout
- Sticky navbar with backdrop blur on scroll
- Subtle accent color (muted gold)

## Notes

- YouTube embed URL needs to be added in `components/Film.tsx`
- Contact email and social media links are placeholders
